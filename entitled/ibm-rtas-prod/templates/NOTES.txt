If you need to login to keycloak browse to https://{{ .Values.global.ibmRtasIngressDomain }}/auth/admin
The username is {{ .Values.keycloak.keycloak.username }} and the password can be retrieved with:
 $ {{ .Values.global.cmdKubectl }} get secret -n {{ .Release.Namespace }} {{ .Release.Name }}-keycloak-postgresql -o jsonpath="{.data.password}" | base64 --decode; echo

Otherwise you can check the status, to ensure pods have settled, with:
 $ watch {{ .Values.global.cmdKubectl }} get pods -n {{ .Release.Namespace }}

The application will then be available at https://{{ .Values.global.ibmRtasIngressDomain }}

A basic test of the installed software may then be done using the command:
 $ helm test {{ .Release.Name }} -n {{ .Release.Namespace }}
Note: Depending on your connection speed this may fail when first run while additional images are pulled.
