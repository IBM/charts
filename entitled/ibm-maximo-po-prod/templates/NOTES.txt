Maximo Production Optimization On-Premises startup will take average 3~5 minutes with sequence.

Please make use of the following ICP's OIDC Authentication endpoints to enforce authentication.
- ICP: https://{{ .Values.global.masterHost }}:{{ .Values.global.masterPort }}
- OIDC: https://{{ .Values.global.masterHost }}:{{ .Values.global.oidcPort }}

Please login as ICP Cluster Admin user to run the following two command to finish the OIDC registration.
- OAUTH2_CLIENT_REGISTRATION_SECRET=$(kubectl get secret platform-oidc-credentials -o yaml -n kube-system | grep OAUTH2_CLIENT_REGISTRATION_SECRET: | awk '{print $2}')
- kubectl exec -it <FULL_POD_NAME_OF_TS> -- /po-config/oidc_reg.sh $OAUTH2_CLIENT_REGISTRATION_SECRET
  * use "kubectl get pods" or "helm status {release name} --tls" to list all Pod names
  * pick any Pod whose name starts with"{release name}-po-ts" to run the above script.
